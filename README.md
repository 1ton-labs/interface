## Get Started

### Requirements

- `react` 18.2.0
- `next` 12.3
- `tailwindcss` 3.2.4
- `typescript` 4.9.5
- `firebase` 8.10.1
- `ton` 13.1.0

### Setup

Install the dependencies

```sh
npm install
```

Start the development server

```sh
npm run dev:ton
```

Build with production mode

```sh
npm run build
```
